The new email account must be created by following the email provider's guidelines.

 Subsequently, the app password for Tête à Tête has to be created by following these steps:

 In the account's Security Settings:

(a) Turn on 2-step Verification,

(b) Search for "App Passwords" and create a new app password for Tête à Tête. The produced 16-digit string (without any spaces) is the app password for Tête à Tête.
